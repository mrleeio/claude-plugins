# ruby-lsp

Ruby language server for Claude Code, providing code intelligence and diagnostics.

## Features

- Diagnostics (errors and warnings via RuboCop)
- Go-to-definition and hover documentation
- Code completion
- Find references
- Symbol search
- Semantic highlighting

## Supported Extensions

- `.rb`, `.rbi` - Ruby source files
- `.rake` - Rake tasks
- `.gemspec` - Gem specifications
- `.ru` - Rack config files
- `.erb` - ERB templates
- `.builder`, `.jbuilder`, `.rabl` - View builders
- `.thor` - Thor CLI scripts
- `.podspec` - CocoaPods specs
- `.irbrc`, `.pryrc` - REPL config files

## Installation

### Via Homebrew (macOS)

```bash
brew install ruby-lsp
```

### Via RubyGems

```bash
gem install ruby-lsp
```

## Requirements

- Ruby 3.0 or higher
