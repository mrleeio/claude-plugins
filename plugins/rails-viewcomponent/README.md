# rails-viewcomponent

ViewComponent conventions and API reference for building reusable Rails view components.

## What This Plugin Does

When working on Rails components, Claude will:

- Follow ViewComponent file structure and naming conventions
- Use slots correctly (renders_one, renders_many, polymorphic)
- Set up component previews for documentation
- Write proper component unit tests

## Features

- Component structure and initialization patterns
- Slots API (passthrough, component, lambda, polymorphic)
- Preview system for component documentation
- Testing with render_inline and Capybara matchers
